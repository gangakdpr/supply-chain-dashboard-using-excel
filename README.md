# supply-chain-dashboard-using-excel
Supply Chain Analysis Project – Excel-based supply chain data cleaning, analysis, and interactive dashboard for monitoring sales, inventory, suppliers, shipping performance, and operational efficiency.
Supply Chain Analysis Project (Excel)

This project is an Excel-based Supply Chain Analytics dashboard that analyzes supply chain operations and business performance using raw and cleaned datasets.

Project Overview

The workbook includes data cleaning, analysis, and visualization to help identify trends and improve supply chain decision-making.

Key Areas Analyzed
Product performance
Sales and revenue analysis
Inventory and stock level monitoring
Supplier performance evaluation
Shipping cost and delivery analysis
Lead time analysis
Defect rate analysis
Customer demographic insights
Workbook Structure
supply_chain_data

Contains the original supply chain dataset.

cleaned data

Processed and cleaned dataset prepared for analysis.

Analysis

Contains calculations, summaries, and performance metrics.

Dashboard

Interactive dashboard that visualizes key supply chain KPIs.

Tools Used
Microsoft Excel
Excel formulas
Pivot Tables
Charts and visualizations
Data cleaning techniques
Objective

The objective of this project is to analyze supply chain performance and provide actionable insights to improve operational efficiency, inventory management, supplier performance, and shipping processes.
