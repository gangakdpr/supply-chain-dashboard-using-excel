# 📦 Supply Chain Performance Dashboard (Excel)

A professional **Supply Chain Analytics Dashboard** built entirely in **Microsoft Excel** to analyze revenue, supplier quality, shipping performance, and profitability across different product categories.

This project transforms raw supply chain data into actionable business insights using **Pivot Tables, Pivot Charts, KPIs, Slicers, and Interactive Dashboards**.


---

# 🚀 Project Overview

The dashboard helps stakeholders monitor supply chain performance and identify areas for improvement.

It answers key business questions such as:

- Which product category generates the highest revenue?
- Which supplier has the highest defect rate?
- Which transportation mode delivers the fastest?
- Which products generate the most profit?
- What are the overall supply chain trends?

The primary objectives of this project are:

# 📊 Dashboard KPIs

| KPI | Description |
|-----|-------------|
| 💰 Total Revenue | Total revenue generated from all product categories |
| ⚠️ Average Defect Rate | Average supplier defect percentage |
| 🚚 Average Shipping Time | Average delivery duration |
| 📈 Profitability | Profit generated by each product category |

## 2️⃣ Supplier Quality Analysis

## 3️⃣ Shipping Performance Analysis

Compared transportation methods:
